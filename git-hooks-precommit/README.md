# Pre-commit хуки (Husky + lint-staged)

## Что делает

Автоматически запускает линтеры и форматтеры (ESLint, Prettier) на **закоммиченных файлах** перед каждым коммитом. Исправляет ошибки и форматирует код автоматически.

## Как работает

- Работает только с **за-staged** файлами (быстро)
- Не блокирует коммит при ошибках форматирования (всё чинится автоматически)
- Настраивается под любой стек: Vue, React, Node и т.д.

## Как вызвать

Этот скилл загружается автоматически, когда вы просите:

- "добавь pre-commit хуки"
- "настрой husky"
- "линтинг перед коммитом"
- "автоформатирование при коммите"

Или вызовите вручную через команду `/pre-commit` (если поддерживается).
