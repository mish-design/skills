# ai-skills
Набор часто используемых правил и скилов для AI

Подходит для `claude code`, `cursor`, `opencode` т.к. они имеют общий пусть до скиллов
`.claude/skills/<name>/SKILL.md`

## Как установить 

1. откройте ваш проект
2. в корне проекта должна быть папка `.claude` (_добавьте ее если еще нет_)
3. перейдите в `.claude` и склонируйте этот репозиторий `git clone`
