# ai-skills
Набор часто используемых правил и скилов для AI

## Как установить 

1. склонируйте этот репозиторий себе в проект `git clone`
2. добавьте директорию со скиллами в `gitignore`
3. переименуйте файлы если требуется для понимания той моделью, которую вы используйте. Или добавьте ссылку на них в конфиг модели.
