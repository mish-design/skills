# Axios Setup

## Что делает

Создаёт готовый к продакшену API-слой на базе Axios:

- **Interceptors** для автоматического добавления auth-токенов
- **Retry** при сетевых ошибках и 5xx
- **Типизированные** запросы (TypeScript)
- **Централизованная** обработка ошибок
- Гибкая система хранения токенов (localStorage/куки/memory)

## Как работает

Скилл задаёт вопросы о проекте:

1. Где хранить токен? (localStorage / куки / memory / async)
2. Как обновлять токен? (без обновления / httpOnly cookie / endpoint)
3. Base URL API
4. Количество retry попыток
5. Уровень обработки ошибок

Затем генерирует `src/utils/api.ts` с нужной конфигурацией.

## Как вызвать

Этот скилл загружается автоматически, когда вы просите:

- "настрой axios"
- "добавь API слой"
- "настрой auth токены"
- "создай API клиент"
- "настрой interceptors для токена"

Или вызовите вручную через команду `/axios-setup` (если поддерживается).
