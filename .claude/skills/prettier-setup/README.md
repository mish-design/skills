# Prettier Setup

## Что делает

Настраивает Prettier — инструмент для автоматического форматирования кода. Устанавливает зависимости, создаёт конфиг и добавляет скрипт в `package.json`.

## Как работает

- Устанавливает `prettier` как dev-зависимость
- Создаёт `.prettierrc` с разумными настройками по умолчанию
- Добавляет скрипт `format` в package.json

## Как вызвать

Этот скилл загружается автоматически, когда вы просите:

- "настрой Prettier"
- "добавь форматирование"
- "подключи prettier к проекту"
- "установи code formatting"
