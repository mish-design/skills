# Pre-push AI-ревью

## Что делает

Перед каждым `git push` отправляет diff изменений в LLM (OpenAI-совместимый API) и генерирует Markdown-отчёт `AI_REVIEW.md` с рекомендациями по улучшению кода.

## Как работает

- Собирает изменения из `src/` (.vue, .js, .ts)
- Перед отправкой в AI проверяет наличие линтеров и запускает их (улучшает качество ревью)
- **Не блокирует push** — даже если AI недоступен, push продолжится
- Ограничивает размер запроса (защита от переполнения контекста)
- Не отправляет секреты (базовая редаткация)

## Как вызвать

Этот скилл загружается автоматически, когда вы просите:

- "добавь AI ревью перед пушем"
- "настрой pre-push хук с LLM"
- "автоматический код-ревью при пуше"
- "интеграция с OpenAI для проверки кода"

Или вызовите вручную через команду `/pre-push` (если поддерживается).
