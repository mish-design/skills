# i18n Setup

## Что делает

Настраивает **интернационализацию (i18n)** для React (react-i18next) или Vue (vue-i18n):

- **Типизированные переводы** — TypeScript автодополнение ключей
- **Ленивая загрузка** — переводы подгружаются по мере необходимости
- **SSR поддержка** — для Next.js/Nuxt
- **Автодетекция языка** — определяет язык браузера
- **Компоненты для смены языка** — готовый language switcher

## Как работает

1. Определяет фреймворк (React/Vue) и пакетный менеджер (npm/yarn/pnpm)
2. Устанавливает нужные зависимости
3. Создает структуру `locales/` с JSON-файлами переводов
4. Генерирует TypeScript типы для type-safe переводов
5. Создает конфигурацию i18n и интеграцию с фреймворком
6. Добавляет готовые компоненты (language switcher) и хуки

## Особенности

- **React**: react-i18next + TypeScript типы
- **Vue**: vue-i18n v8/v9 + Composition API helpers
- **Next.js**: next-i18next для App Router/Pages Router
- **Lazy loading**: переводы загружаются только для активного языка
- **Type-safe**: все ключи переводов проверяются TypeScript

## Как вызвать

Этот скилл загружается автоматически, когда вы просите:

- "настрой i18n"
- "добавь поддержку нескольких языков"
- "настрой интернационализацию"
- "добавь переводы"
- "настрой react-i18next/vue-i18n"

Или вызовите вручную через команду `/i18n-setup` (если поддерживается).
