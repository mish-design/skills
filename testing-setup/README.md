# Настройка тестирования (Vitest/Jest)

## Что делает

Устанавливает и настраивает универсальный фреймворк для тестирования с поддержкой покрытия кода. Поддерживает Vitest (рекомендуется) и Jest.

## Как работает

- **Автоматическое определение** — проверяет существующий фреймворк в package.json
- **Универсальность** — работает с React, Vue, Svelte и чистым JS/TS
- **Testing Library** — DOM-тестирование через @testing-library
- **Покрытие кода** — встроенная поддержка coverage через v8

## Как вызвать

Этот скилл загружается автоматически, когда вы просите:

- "настрой тестирование"
- "добавь тесты"
- "установи vitest/jest"
- "настрой testing library"
- "добавь unit-тесты"

## Поддерживаемые фреймворки

| Фреймворк | Когда использовать |
|-----------|-------------------|
| **Vitest** | Новые проекты, Vite, быстрая разработка |
| **Jest** | Существующие проекты, большое комьюнити |
